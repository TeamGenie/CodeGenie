##Doel:
Website die docenten toelaat om studenten aan self-assesment te doen in de lessen programmeren in 1ea of 2ea. (Het concept is in principe perfect toepasbaar in andere lessen) 

De website houdt alle ooit gevraagde oefeningen bij en studenten kunnen punten verdienen door oefeningen te maken Ã©n aan te duiden als klaar. Per oefening kunnen ze aangeven welke onderdelen ze wel of niet hebben gedaan zodat een deelscore kan berekend worden. De student kan z'n eigen progressie vergelijken met die van de gemiddelde klasprogressie.  

De docent kan via de website vlot feedback geven en scores goed/afkeuren.

Student heeft hierdoor een quasi '"live" inzicht in hoe goed of slecht hij bezig is.
